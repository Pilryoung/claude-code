# Claude Code 신규 패치 — 2026-05-12

## v2.1.139 (released 2026-05-11)
- **에이전트 뷰 추가 (Research Preview)**: `claude agents` 명령으로 실행 중·대기·완료된 모든 세션을 한 목록에서 확인 가능
- **`/goal` 명령 추가**: 완료 조건을 설정하면 Claude가 목표 달성까지 여러 턴에 걸쳐 자동 작업 수행, 경과 시간·턴 수·토큰을 오버레이로 표시
- **`/scroll-speed` 명령 추가**: 마우스 휠 스크롤 속도를 실시간 미리보기로 조정
- **훅 개선**: `args: string[]` exec 형식으로 셸 없이 직접 명령 실행, `PostToolUse`에 `continueOnBlock` 옵션 추가(거부 이유를 Claude에게 전달하고 턴 계속 진행)
- **MCP stdio 서버가 `CLAUDE_PROJECT_DIR` 환경변수 수신**, 플러그인 설정에서 `${CLAUDE_PROJECT_DIR}` 경로 참조 가능
- **API 키 설정 시 원격 기능 비활성화**: `ANTHROPIC_API_KEY` 등 설정 시 Remote Control·`/schedule`·claude.ai MCP 커넥터 자동 비활성화
- **버그 수정**: 인증 명령 데드락, `autoAllowBashIfSandboxed`의 셸 확장(`$VAR`, `$(cmd)`) 미승인, 훅 실행 중 인터랙티브 프롬프트 깨짐, HTTP/SSE MCP 스트림 무제한 메모리 증가(16 MB 캡 적용), `Skill(name *)` 와일드카드 매칭 오류, `~/.claude/settings.json` 심링크 핫리로드 미감지 등 40여 건 수정
- **[VSCode]** Cmd/Ctrl+Shift+T로 최근 닫은 세션 탭 다시 열기 (`claudeCode.enableReopenClosedSessionShortcut`로 설정 가능)
- **Cursor·VS Code 1.92–1.104 마우스 휠 스크롤 속도 수정**, CJK/이모지 셀 너비 오버플로우 및 퍼지 매칭 하이라이팅 오류 수정
