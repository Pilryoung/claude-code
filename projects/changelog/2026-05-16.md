# Claude Code 신규 패치 — 2026-05-16

## v2.1.143 (released 2026-05-15)
- 플러그인 의존성 강제 적용: `claude plugin disable` 명령이 다른 활성 플러그인이 의존하는 경우 비활성화를 거부하고 비활성화 순서 힌트를 제공; `plugin enable`은 전이 의존성을 자동으로 활성화
- `worktree.bgIsolation: "none"` 설정 추가 — 워크트리 사용이 어려운 저장소에서 백그라운드 세션이 `EnterWorktree` 없이 작업 디렉토리를 직접 편집 가능
- Windows/WSL 다수 버그 수정: `claude agents`에서 우클릭 붙여넣기, 스트리밍 중 방향키 입력 불응답, PowerShell 프로세스 중복 생성 등
- 스톱 훅 8회 연속 차단 시 경고 후 강제 종료로 무한루프 방지 (`CLAUDE_CODE_STOP_HOOK_BLOCK_CAP`으로 조정 가능); `/bg` 분리 시 `--fallback-model`, `--mcp-config`, `--settings` 등 옵션이 백그라운드 세션에 유지되도록 개선
- 손상된 `.credentials.json`으로 CLI 시작 시 멈추던 버그 수정; `NO_COLOR`/`FORCE_COLOR` 환경변수가 Claude Code UI 색상에도 영향을 주던 버그 수정(하위 프로세스에만 적용되도록 변경)
