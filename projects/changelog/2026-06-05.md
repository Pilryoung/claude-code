# Claude Code 신규 패치 — 2026-06-05

## v2.1.163 (released 2026-06-04)
- 관리형 설정에 `requiredMinimumVersion`/`requiredMaximumVersion` 추가 — 허용 버전 범위 밖이면 Claude Code 실행 거부 및 승인된 버전으로 안내
- `/plugin list` 명령어 신규 추가 (`--enabled`/`--disabled` 필터 지원)
- `/btw` 답변에 `c` 단축키로 raw markdown 클립보드 복사 기능 추가
- 훅 개선: Stop/SubagentStop 훅에서 `hookSpecificOutput.additionalContext` 반환으로 오류 처리 없이 Claude에게 피드백 전달 가능
- `claude -p` 사용 시 백그라운드 명령이 종료되지 않으면 영구 행 걸리던 버그 수정 (stdin 닫힌 뒤 ~5초 후 백그라운드 셸 종료)
- Bazel·EDR 보호 Go 워크플로우에서 `$TMPDIR`이 샌드박스 외 명령에도 덮어써지던 회귀 버그 수정 (2.1.154 도입)
- 훅 `if: "Bash(...)"` 조건이 `$()` 또는 `$VAR` 포함 명령 전체에 잘못 매칭되던 버그 수정
