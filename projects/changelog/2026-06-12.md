# Claude Code 신규 패치 — 2026-06-12

## v2.1.174 (released 2026-06-12)
- 신기능: 전체화면 모드에서 마우스 휠 스크롤 가속을 비활성화하는 `wheelScrollAccelerationEnabled` 설정 추가
- 버그픽스: `/model` 피커에서 기본값이 해석되는 모델 패밀리가 숨겨지던 오류 수정 — Max/Team Enterprise는 Opus, Pro/Team은 Sonnet, PAYG API는 Opus로 별도 행 표시

- 버그픽스: 백그라운드 세션이 다른 세션의 `ANTHROPIC_*` 환경변수(게이트웨이 URL, 커스텀 헤더, `/model` 별칭)를 상속하던 버그 수정
- 기능 개선: [VSCode] `/usage` 다이얼로그에 캐시 미스·장문 컨텍스트·서브에이전트·스킬/에이전트/플러그인별 상세 사용량 어트리뷰션 추가 (지난 24h/7d 기준)
