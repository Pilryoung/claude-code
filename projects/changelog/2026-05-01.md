# Claude Code 신규 패치 — 2026-05-01

## v2.1.126 (released 2026-05-01)
- `claude project purge [path]` 명령 추가 — 프로젝트의 전사본·작업·파일 히스토리·설정을 일괄 삭제 (`--dry-run`, `-y/--yes`, `-i/--interactive`, `--all` 옵션 지원)
- `claude auth login`이 WSL2·SSH·컨테이너 환경에서 브라우저 콜백 없이 터미널에 직접 OAuth 코드를 붙여넣어 인증 가능
- **[보안]** `allowManagedDomainsOnly`/`allowManagedReadPathsOnly` 설정이 상위 관리 소스에 `sandbox` 블록 없을 때 무시되는 보안 취약점 수정
- 대형 이미지(2000px 초과) 붙여넣기 시 세션이 끊기는 버그 수정 — 붙여넣기 시 자동 축소 처리, 히스토리 내 초과 이미지도 자동 제거 후 재시도
- Mac 슬립 후 및 백그라운드·원격 세션에서 "Stream idle timeout"으로 세션이 잘못 중단되는 버그 수정
- Windows에서 no-flicker 모드 시 한국어·일본어·중국어 텍스트가 깨진 문자로 표시되는 버그 수정; PowerShell 7 감지 방식 개선 (Microsoft Store·MSI 등)
