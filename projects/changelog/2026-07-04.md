# Claude Code 신규 패치 — 2026-07-04

## v2.1.200 (released 2026-07-03)
- `AskUserQuestion` 대화창이 기본적으로 자동 계속 진행하지 않도록 변경; idle timeout은 `/config`에서 별도 설정 가능
- 권한 모드 명칭 "default" → "Manual"로 통일 (CLI, `--help`, VS Code, JetBrains 전체 반영)
- 백그라운드 세션·에이전트 다수 버그 수정: 슬립/웨이크 후 중단, stale `daemon.lock` PID 재사용, 구버전 바이너리의 데몬 탈취, 소켓 인증 토큰 탈락 등
- tmux 3.4+에서 동기화 출력(synchronized terminal output) 활성화로 렌더링 깜빡임 수정
- 스크린 리더 접근성 개선: 장식용 글리프 숨김, 중첩 테이블을 `Header: value.` 형식으로 읽기

## v2.1.199 (released 2026-07-02)
- 슬래시 스킬 중첩 호출(`/skill-a /skill-b do XYZ`) 시 선두 스킬 최대 5개 동시 로드 지원 (기존: 첫 번째만)
- SSL 인증서 오류(TLS 검사 프록시·만료 인증서)를 재시도 소모 없이 즉시 실패 처리하고 수정 힌트 표시
- 스트리밍 중 서버 오류 시 수신된 부분 응답 보존; 서브에이전트 rate limit·서버 오류를 성공으로 잘못 보고하던 버그 수정
- 백그라운드 에이전트 다수 버그 수정: Linux에서 약 50초마다 자가 종료, SSH macOS 냉시작 실패(2.1.196 회귀), `claude stop`이 재스폰으로 무효화되는 문제 등
- 구독자 대상 일시적 429 rate limit 오류 자동 재시도; `CLAUDE_CODE_RETRY_WATCHDOG`으로 기본 재시도 횟수 300회 상향, `CLAUDE_CODE_MAX_RETRIES` 15회 상한 제거
