- 로그인 만료 임박 경고 추가 — 백그라운드 세션이 끊기기 전 재인증 시점을 알림; 수동 권한 모드 활성화 시 풋터에 회색 ⏸ 배지 표시
- macOS에서 백그라운드 에이전트 세션 열기·전환 시 15~20초 지연 수정 (v2.1.196 회귀, 잘못된 저메모리 감지 원인)
- 데몬 세션 토큰 만료 시 백그라운드 세션이 응답 불가 상태로 영구 고착되던 문제 자동 복구로 해결

- 바이너리 크기 약 7 MB·시작 메모리 약 7 MB 절감 (대형 번들 의존성 지연 로드)
