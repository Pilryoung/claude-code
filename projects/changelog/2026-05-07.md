# Claude Code 신규 패치 — 2026-05-07

## v2.1.132 (released 2026-05-07)
- 신규: `CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN=1` 환경변수 추가 — 풀스크린 대체 렌더러를 끄고 터미널 기본 스크롤백 유지 가능
- 버그픽스: 외부 SIGINT(IDE 중지 버튼, `kill -INT`) 수신 시 graceful shutdown이 실행되지 않아 터미널 모드가 복원되지 않던 문제 수정
- 버그픽스: 랩탑 슬립/복귀 또는 Ctrl+Z·`fg` 후 풀스크린 모드에서 빈 화면이 표시되던 문제 수정
- 버그픽스: stdio MCP 서버가 stdout에 비프로토콜 데이터를 쓸 때 메모리가 무한 증가(10GB+ RSS)하던 누수 수정
- 버그픽스: `--permission-mode` 플래그가 plan-mode 세션 재개(`-p --continue`/`--resume`) 시 무시되던 문제 수정
- 버그픽스: Bedrock 및 Vertex에서 `ENABLE_PROMPT_CACHING_1H` 설정 시 발생하던 400 오류 수정
