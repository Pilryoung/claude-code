# Claude Code 신규 패치 — 2026-06-04

## v2.1.162 (released 2026-06-04)
- **`claude agents` 대규모 버그픽스**: 와이드 터미널에서 상태 텍스트/세션명 잘림 수정, Ctrl+V 이미지 붙여넣기 작동, attach 시 즉시 목록 복귀 버그 수정, 전송 실패 답장 큐잉 처리, 백그라운드 서비스 재시작 후 대화 유실 방지
- **신기능**: `claude agents --json`에 `waitingFor` 필드 추가 (세션 블록 원인 표시); 슬래시 명령어 자동완성 클릭 시 즉시 실행 대신 입력창에 채워넣기(Enter로 실행); Remote Control이 고정 Footer pill로 표시
- **권한·MCP 버그픽스**: WebFetch 명시적 deny/ask/allow 규칙이 사전 승인 도메인에도 정상 적용; Windows 백슬래시·대소문자 경로에서 권한 규칙 미매칭 수정; MCP per-server timeout이 1000ms 미만일 때 1초 watchdog으로 올라가던 버그 수정; LSP `workspaceSymbol` 결과 미반환 수정
- **시작 화면 간소화**: 알림이 심각도별로 그룹화, 세션 정보·공지 한 줄 통합; "Claude in Chrome enabled"·"marketplace installed" 시작 메시지 제거; 실패 턴이 컴팩트한 경고 줄로 표시
- **Devin Desktop 리브랜딩 반영**: `/ide`, `/terminal-setup`, `/scroll-speed` 메뉴에서 Windsurf → Devin Desktop으로 명칭 변경
