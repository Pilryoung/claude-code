# Claude Code 신규 패치 — 2026-05-08

## v2.1.133 (released 2026-05-07)
- `worktree.baseRef` 설정 추가(`fresh` | `head`): 워크트리/에이전트 격리 생성 시 기준 브랜치를 `origin/<default>`(기본값) 또는 로컬 `HEAD` 중 선택 가능 — 기본값 `fresh`로 `EnterWorktree`가 2.1.128 이전의 origin 기준 동작으로 복귀
- Hooks 및 Bash 도구 명령에서 `effort.level` JSON 필드·`$CLAUDE_EFFORT` 환경 변수로 현재 effort 레벨 참조 가능
- 메모리 압박 시 예비 백그라운드 워커 자동 해제로 메모리 사용량 개선
- refresh-token 경쟁 조건으로 공유 자격 증명이 지워지며 병렬 세션이 모두 401 오류에서 중단되던 버그 수정
- `HTTP(S)_PROXY` / `NO_PROXY` / mTLS 설정이 MCP OAuth 전체 흐름(discovery·동적 클라이언트 등록·token 교환·갱신)에 적용되지 않던 버그 수정
