# Claude Code 신규 패치 — 2026-06-03

## v2.1.161 (released 2026-06-02)
- **[신기능]** `OTEL_RESOURCE_ATTRIBUTES` 값을 메트릭 데이터포인트 레이블로 포함 — 팀·레포 등 커스텀 차원으로 사용량 통계 분류 가능
- **[신기능]** 병렬 툴 호출 시 Bash 명령 하나가 실패해도 같은 배치의 다른 호출이 취소되지 않고 각각 독립 결과 반환
- **[버그픽스/보안]** `claude mcp` list·get·add 명령이 터미널에 인증 헤더·URL 시크릿을 노출하던 문제 수정 (`${VAR}` 확장 차단 및 크리덴셜 마스킹)
- **[버그픽스]** `forceLoginOrgUUID`/`forceLoginMethod` 정책이 Bedrock·Vertex·Foundry·Mantle 등 서드파티 세션을 차단하던 v2.1.146 이후 회귀 수정
- **[버그픽스]** 백그라운드 서브에이전트 출력이 `claude -p --output-format text/json`의 stdout을 오염하던 문제 수정
- **[성능]** 레이아웃 엔진 JIT 컴파일 프로파일 안정화로 터미널 렌더링 성능 향상; 대형 파일 쓰기 렌더링 성능도 개선
