# Claude Code 신규 패치 — 2026-05-14

## v2.1.141 (released 2026-05-14)

### 신규 기능 & 개선
- `terminalSequence` 훅 필드 추가 — 제어 터미널 없이도 훅에서 데스크탑 알림·윈도우 타이틀·벨 신호 전송 가능
- `CLAUDE_CODE_PLUGIN_PREFER_HTTPS` 환경변수 추가 — GitHub SSH 키가 없는 환경에서 플러그인을 HTTPS로 클론
- `ANTHROPIC_WORKSPACE_ID` 환경변수 추가 — 워크로드 ID 페더레이션 시 발행 토큰을 특정 워크스페이스로 범위 지정
- `claude agents --cwd <경로>` 옵션 추가 — 지정 디렉토리의 에이전트 세션만 표시
- `/feedback`에서 최근 24시간·7일 세션 포함 가능 — 여러 세션에 걸친 이슈를 한 번에 보고 지원
- Rewind 메뉴에 "여기까지 요약" 옵션 추가 — 최근 대화를 보존하면서 이전 컨텍스트 압축
- 배경 에이전트(`/bg`, `←←`) 실행 시 기본 권한 모드 대신 현재 권한 모드 유지
- 스피너 피드백 개선 — 긴 사고 중 10초 경과 시 호박색으로 변해 대기 상태를 시각화
- 플러그인 메뉴 탐색 개선 — `→`/Tab으로 탭 전환, `↑`으로 탭 스트립 이동, 전체화면 모드에서 탭 헤더·검색창 클릭 가능

### 버그 수정
- Bedrock/Vertex/Foundry 등 타사 환경에서 백그라운드 사이드 쿼리가 사용 불가 Haiku 모델 ID를 전송하던 문제 수정 (메인 루프 모델로 폴백)
- `/model` 변경이 다른 동시 실행 세션의 autocompact 임계값을 변경하던 문제 수정
- 마크다운 테이블 셀 래핑 시 테두리 그리드 대신 세로 키-값 레이아웃으로 폴백되던 회귀 수정 (v2.1.136 이후)
- Ctrl+C가 vim INSERT/VISUAL 모드에서 실행 중인 작업을 중단하지 못하던 문제 수정
- `chat:submit` 외 대체 키 바인딩(meta+enter, ctrl+enter 등)이 `enter`를 newline으로 재바인딩했을 때 동작하지 않던 문제 수정
- Remote Control MCP 커넥터 전체가 세션 토큰 갱신 시 401을 반환하던 문제 수정
- Windows에서 Alt+V 이미지 붙여넣기가 "이미지를 찾을 수 없음"을 표시하던 문제 수정
- Linux에서 glibc·musl 플랫폼 패키지 동시 설치 시 SDK "Claude Code native binary not found" 오류 수정
- MCP HTTP/SSE 서버가 연결 시 403을 반환할 경우 "failed" 대신 "needs auth"로 명확히 표시
- 원격 MCP 서버가 선택적 server-events 스트림 재연결 실패 시 불필요하게 연결 끊기는 문제 수정 (POST로 도구 호출 계속)
- Bedrock: 앰비언트 AWS 자격증명이 해결되어도 `awsCredentialExport`가 항상 실행되도록 수정 — 크로스 계정 액세스 인증 문제 해결
