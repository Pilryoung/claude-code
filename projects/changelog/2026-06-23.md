# Claude Code 신규 패치 — 2026-06-23

## v2.1.186 (released 2026-06-22)
- **[신기능]** `claude mcp login/logout <name>` CLI 명령 추가 — 대화형 `/mcp` 메뉴 없이 터미널에서 바로 MCP 서버 인증 가능 (SSH 환경의 `--no-browser` stdin 리다이렉트 지원 포함)
- **[신기능]** `!` bash 명령 실행 시 Claude가 출력을 자동 분석·응답; 이전 동작(컨텍스트만 추가)으로 되돌리려면 settings.json에 `"respondToBashCommands": false` 설정
- **[신기능]** 백그라운드 서브에이전트의 권한 요청이 자동 거부 대신 메인 세션에 표시되도록 변경 — 어느 에이전트가 요청 중인지 다이얼로그 표시, Esc로 해당 도구만 거부 가능
- **[버그픽스]** 슬립 복귀 후 스트리밍 요청에서 "Content block not found" 및 JSON 파싱 오류 발생하던 문제 수정
- **[버그픽스]** 백그라운드 에이전트 실행 중 Esc / Ctrl+C가 응답하지 않던 문제 수정
- **[버그픽스]** `~~취소선~~` 문법이 리터럴 틸드로 표시되던 렌더링 버그 수정
- **[버그픽스]** `Agent(type)` 거부 규칙 및 허용 유형 제한이 명명된 서브에이전트 스폰에 적용되지 않던 보안 버그 수정
- **[버그픽스]** Workflow `agent({schema})` 서브에이전트가 스키마 검증 실패 시 무한 루프에 빠지던 문제 수정 (5회 시도 후 중단)
- **[변경]** `CLAUDE_CODE_MAX_RETRIES` 상한값 15로 제한 — 무인 세션에는 `CLAUDE_CODE_RETRY_WATCHDOG` 사용 권장
- **[변경]** `/review <pr>` 명령이 `/code-review medium`과 동일한 검토 엔진 사용하도록 통일
