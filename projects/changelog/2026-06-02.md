# Claude Code 신규 패치 — 2026-06-02

## v2.1.160 (released 2026-06-02)
- **보안 강화**: 쉘 시작 파일(`.zshenv`, `.zlogin`, `.bash_login`) 및 빌드 도구 설정 파일(`.npmrc`, `.yarnrc*`, `bunfig.toml`, `.bazelrc`, `.pre-commit-config.yaml`, `.devcontainer/` 등)에 쓰기 전 확인 프롬프트 추가 — 의도치 않은 코드 실행 방지
- **UX 개선**: `grep`/`egrep`/`fgrep`으로 단일 파일 조회 후 별도 Read 없이 바로 편집 가능 (read-before-edit 조건 충족으로 처리)
- **WSL·Windows 버그 수정**: WSL에서 선택 복사 시 클립보드 미반영 문제 해결(OSC 52 → PowerShell interop), `claude rm` 후 Windows 디렉터리 삭제 불가 문제 해결, `file:///C:/...` 링크 경로 깨짐 수정
- **백그라운드 에이전트 안정성**: 세션 복원 시 대화 기록 소실 및 최초 프롬프트 재실행 버그, `--bg` "socket missing" 오류, Windows에서 Esc·화살표키 무응답, CJK IME 입력창 위치 오류(좌하단→커서 위치), 음성 모드 비ASCII 경로 연결 실패 등 다수 수정
- **키워드 변경 및 제거**: 동적 워크플로 트리거 키워드 `workflow` → `ultracode` 변경(보라색 강조 표시), `CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE` 환경변수 제거, JetBrains 플러그인 설치 안내 제거
