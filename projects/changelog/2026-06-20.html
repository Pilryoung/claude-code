🔄 Claude Code 신규 패치

v2.1.183 released 2026-06-19

  • 안전성 강화: Auto 모드에서 git reset --hard, git checkout -- ., git clean -fd, git stash drop 등 파괴적 git 명령을 사용자가 명시적으로 요청하지 않은 경우 차단; git commit --amend는 해당 세션에서 에이전트가 생성하지 않은 커밋에 대해 차단; terraform destroy/pulumi destroy/cdk destroy는 대상 스택을 명시 요청한 경우에만 허용
  • 신규 설정 추가: attribution.sessionUrl 설정으로 웹·Remote Control 세션에서 커밋 및 PR에 삽입되는 claude.ai 세션 링크를 생략 가능; /config --help로 /config key=value 단축키 전체 목록 조회 지원; /config 토글에서 Enter·Space로 값 변경, Esc로 저장 후 닫기로 동작 변경
  • 버그 수정: 서브에이전트 스폰 및 세션 타이틀 생성 시 thinking.disabled.display: Extra inputs are not permitted 400 오류 수정; 서브에이전트에서 WebSearch가 빈 결과를 반환하던 문제 수정
  • 버그 수정: 모델이 thinking 블록만 반환할 때 출력 없이 조용히 완료되던 문제 수정 (이제 재프롬프트 1회 수행); 스케줄 태스크·웹훅 트리거가 키보드 입력으로 처리되어 auto 모드에서 액션 승인이나 세션 타이틀 설정을 트리거하던 버그 수정
  • 버그 수정: Windows Terminal에서 중첩 서브에이전트 부하 시 전체화면 TUI 깨짐(statusline 중간 위치, 스피너 행 중복, 텍스트 병합) 수정; tmux 팀메이트 팬이 쉘 rc 파일 초기화 지연 시 실행 실패하고 에이전트 스폰 중 입력이 새 tmux 팬으로 누출되던 문제 수정
