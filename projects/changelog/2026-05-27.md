# Claude Code 신규 패치 — 2026-05-27

## v2.1.152 (released 2026-05-27)
- `/code-review --fix` 명령이 리뷰 결과를 작업 트리에 직접 적용; `/simplify`가 동일 동작으로 재정의됨
- 스킬·슬래시 커맨드 프론트매터에 `disallowed-tools` 지정 가능 — 해당 스킬 활성화 중 특정 도구를 모델에서 제거
- `SessionStart` 훅에서 `reloadSkills: true` 반환 시 동일 세션 내 훅이 설치한 스킬 즉시 사용 가능; `/reload-skills` 명령으로 수동 재스캔도 지원
- `MessageDisplay` 훅 이벤트 추가 — 어시스턴트 메시지 텍스트를 표시 전 변환·숨김 처리 가능
- 기본 모델 미발견 시 `--fallback-model`로 세션 내 자동 전환 (기존: 매 요청마다 실패)
- Auto 모드 사전 동의 절차 제거 (옵트인 없이 즉시 사용 가능)
- Vim NORMAL 모드에서 `/`키로 역방향 히스토리 검색 지원 (bash/zsh vi-mode와 동일)
- 사고 요약이 최소 3초 이상 표시되며 마크다운 렌더링·10줄 상한 적용 (`Ctrl+O`로 전체 보기)
- 긴 세션에서 터미널 스타일이 점진적으로 깨지던 버그 수정 (렌더러 스타일 풀 재활용)
- 모델·로그인 전환 후 세션이 멈추던 버그 수정 (stale thinking-block 시그니처 선제 제거 + 재시도 안전망)
- 플러그인 MCP 서버가 환경변수만 다른 경우에도 잘못 중복 제거되던 버그 수정
- `cache_creation_input_tokens`가 0으로 잘못 집계되던 버그 수정
