# Claude Code 신규 패치 — 2026-07-07

## v2.1.202 (released 2026-07-06)
- `/config`에 "Dynamic workflow size" 설정 추가 — 워크플로우가 생성하는 에이전트 수를 소/중/대로 조절하는 가이드라인 설정 (강제 상한 아님)
- git worktree가 많은 저장소에서 세션 이름으로 재개하거나 재개 선택기를 열 때 수 분이 걸리고 메모리를 과다 사용하던 문제 수정
- Remote Control(모바일/웹)에서 캡션 없이 보낸 이미지·파일이 무음으로 누락되던 문제 및 명령 전송 시 "Unknown command" 오류 발생 문제 수정
- 프록시·네트워크 중단으로 설치·업데이트 다운로드가 즉시 "aborted"로 실패하던 문제 수정 — 일시적 연결 끊김 시 자동 재시도
- `/review <pr>`를 빠른 단일 패스 리뷰로 복원; 멀티 에이전트 심층 리뷰는 `/code-review <level> <pr#>` 사용
