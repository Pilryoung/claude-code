# Claude Code 신규 패치 — 2026-05-15

## v2.1.142 (released 2026-05-15)
- `claude agents`에 배경 세션 설정용 플래그 8개 신규 추가 (`--model`, `--permission-mode`, `--settings`, `--mcp-config`, `--plugin-dir`, `--effort`, `--add-dir`, `--dangerously-skip-permissions`)
- Fast mode 기본 모델이 Opus 4.6 → Opus 4.7로 변경 (`CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1` 환경변수로 Opus 4.6 고정 가능)
- `MCP_TOOL_TIMEOUT` 설정이 원격 HTTP/SSE MCP 서버에 미반영되어 도구 호출이 항상 60초로 제한되던 버그 수정
- macOS 절전/복귀 후 배경 세션 데몬 재연결 실패 문제 수정 (클럭 점프를 idle 경과 시간으로 오인하는 원인 해결)
- Windows 네트워크 드라이브 작업 디렉터리에서 `claude agents` 데드락 발생 및 시작 중 Ctrl+C 무반응 문제 수정
