# Claude Code 신규 패치 — 2026-06-11

## v2.1.172 (released 2026-06-11)
- **서브에이전트 중첩 실행**: 서브에이전트가 자체 서브에이전트를 최대 5단계까지 스폰 가능해져 복잡한 멀티에이전트 워크플로 지원
- **1M 컨텍스트 영구 정지 버그 수정**: 사용 크레딧 없는 1M 컨텍스트 세션이 영구 정지되던 문제 해결 — 세션이 표준 컨텍스트 한도로 자동 압축됨
- **Amazon Bedrock 리전 자동 인식**: `AWS_REGION` 미설정 시 `~/.aws` 설정 파일에서 리전을 자동으로 읽어 AWS SDK 동작과 일치, `/status`에서 리전 출처 표시
- **성능 및 CPU 개선**: 장시간 대화에서 불필요한 메시지 정규화 제거, 유휴 상태에서 `/goal` 상태 칩 5Hz 재렌더링 제거로 CPU 사용량 감소
- **`availableModels` 및 와일드카드 권한 규칙 버그 수정**: 서브에이전트 모델 오버라이드·어드바이저 모델에 제한 미적용 문제, `WebFetch(domain:*.example.com)` 와일드카드 서브도메인 매칭 및 파일 권한 패턴 와일드카드 처리 버그 수정
- **`/plugin` 마켓플레이스 검색 바 추가**: 마켓플레이스 플러그인 탐색 시 검색 바를 통한 빠른 필터링 지원
- **배경 에이전트 격리 버그 수정**: 사전 워밍된 워커에 디스패치된 배경 에이전트가 다른 디렉토리의 프로젝트 설정(`.mcp.json` 승인, 신뢰 설정)을 읽던 보안 문제 수정
- **[VSCode] PowerShell 도구 호출 표시 수정**: 원시 JSON으로 렌더링되던 PowerShell 도구 호출이 올바른 명령 표시와 권한 다이얼로그로 출력, 셸 출력의 ANSI 이스케이프 코드 제거
