# Claude Code 신규 패치 — 2026-05-05

## v2.1.128 (released 2026-05-04)
- `--plugin-dir`가 디렉토리 외 `.zip` 아카이브 직접 로드 지원 추가
- `/color` 인수 없이 실행 시 랜덤 세션 색상 자동 선택, `/mcp`에서 연결 서버의 도구 수 표시 및 도구 0개 서버 경고 플래그 추가
- `EnterWorktree`가 `origin/<기본 브랜치>` 대신 로컬 HEAD 기준으로 브랜치 생성 — 미푸시 커밋 손실 방지
- `claude -p`에 stdin으로 10 MB 초과 대용량 입력 파이프 시 발생하던 크래시 루프 수정
- 병렬 셸 도구 호출에서 읽기 전용 명령(grep, git diff 등) 실패가 형제 호출을 취소하던 버그 수정
- MCP stdio 서버에 공백·메타문자 포함 인수 전달 시 인수가 손상되던 버그 수정 (`CLAUDE_CODE_SHELL_PREFIX` 설정 환경)
- 서브 에이전트 진행 요약의 프롬프트 캐시 누락 수정 — cache_creation 약 3배 감소
