# Claude Code 신규 패치 — 2026-07-23

## v2.1.218 (released 2026-07-22)
- `/code-review` 명령이 백그라운드 서브에이전트로 실행되도록 변경 — 리뷰 작업이 현재 대화창을 채우지 않으며, 스택된 슬래시 명령 대상으로 정확히 작동
- Windows 경로에서 `\u` 접두사 세그먼트(예: `C:\Users\unicorn`)가 CJK 문자로 깨져 파일에 접근할 수 없던 버그 수정
- 좌측 화살표 키 실수 입력으로 대화가 즉시 삭제되던 버그 수정 — 편집 직후 누를 때 확인 메시지 표시, Esc 키로 백그라운드 대화 복귀 지원
- 특정 터미널에서 멀티라인 붙여넣기 시 줄바꿈이 `j` 문자로 치환되던 버그 수정
- MCP 서버 연결 실패 시 `claude mcp list` 및 `/mcp`에 HTTP 상태·오류 텍스트 표시 추가, 선행/후행 공백 포함 MCP 설정값 경고 추가
- `/deep-research`가 수동 호출 시에만 시작하도록 변경 — Claude가 자동으로 실행하지 않음
