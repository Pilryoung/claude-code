# Claude Code 신규 패치 — 2026-07-21

## v2.1.216 (released 2026-07-20)
- `sandbox.filesystem.disabled` 신규 설정 추가: 파일시스템 격리는 비활성화하면서 네트워크 송신 제어는 유지 가능
- 긴 세션에서 메시지 정규화 비용이 턴 수에 따라 이차적으로 증가하던 성능 이슈 수정 (멀티초 지연 및 느린 재개 현상 해소)
- `.claude` 심링크를 통해 워크플로 저장·예약 작업 쓰기가 프로젝트 외부로 리다이렉트될 수 있는 보안 취약점 수정
- 재개된 백그라운드 에이전트 세션이 기본 에이전트로 초기화되던 버그 수정 (프롬프트·도구 제한 이제 정상 복원)
- 클라우드 세션 컨테이너 재시작 시 진행 중인 메시지가 유실되던 문제 수정 (재개 시 중단된 턴 자동 재실행)
- OAuth 토큰 만료·교체 후 자동 모드에서 명령이 HTTP 401 오류로 거부되던 버그 수정
- [VS Code] 아랍어·히브리어·페르시아어 등 RTL 텍스트를 영어·코드와 혼합 시 렌더링 순서가 뒤바뀌던 문제 수정
