# Claude Code 신규 패치 — 2026-07-02

## v2.1.198 (released 2026-07-01)
- **Chrome GA**: Claude Code가 Chrome 브라우저에서 정식 출시(GA)
- **백그라운드 에이전트 자동 PR**: `claude agents`에서 실행 중인 에이전트가 코드 작업 완료 시 자동으로 커밋·푸시·드래프트 PR 생성 (기존에는 사용자 확인 요청)
- **`/dataviz` 스킬 추가**: 차트·대시보드 디자인 가이드 및 실행 가능한 색상 팔레트 검증 도구 제공
- **Gateway AWS 공급자**: AWS Claude Platform(`anthropicAws`) 업스트림 공급자 추가; 모델 미발견 응답 시 페일오버 체인 자동 전환
- **네트워크 재시도 개선**: 응답 중 일시적 네트워크 단절(ECONNRESET 등) 발생 시 응답 중단 대신 백오프 재시도
- **Explore 에이전트 모델 상속**: 내장 Explore 에이전트가 haiku 고정에서 메인 세션 모델(최대 opus) 상속으로 변경 — 결과 품질 향상
- **확장 사고 상속**: 서브에이전트·컨텍스트 컴팩션이 세션의 extended thinking 설정을 상속하여 위임 작업 품질 향상
- **macOS 로컬 네트워크 수정**: 백그라운드 에이전트 세션에서 로컬 네트워크 호스트 "no route to host" 오류 수정 (Local Network 엔트리 선언)
- **AWS STS 자동 재인증**: AWS/Mantle 세션에서 STS 토큰 만료 시 "Please run /login" 오류 대신 `awsAuthRefresh` 자동 실행
