## v2.1.157 (released 2026-05-30)
- `.claude/skills` 디렉터리의 플러그인이 마켓플레이스 없이 자동 로드되며, `claude plugin init <name>` 명령으로 새 플러그인 스캐폴딩 가능; `/plugin` 인수 자동완성 추가
- `claude agents`: `settings.json`의 `agent` 필드가 디스패치 세션에 반영되고, `EnterWorktree`로 세션 중 Claude 관리 워크트리 간 전환 가능
- 손상된 이미지(0바이트, 깨진 파일) 첨부 시 요청이 중단되는 버그 수정; auto/bypass-permissions 모드에서 sandbox 네트워크 권한 프롬프트가 불필요하게 표시되던 문제 수정
- `--resume`, `--worktree`, tmux 클립보드, VS Code/Cursor/Windsurf 우클릭 붙여넣기 중복 등 다수 버그 수정; `/model` 피커의 잘못된 "신규 버전 있음" 힌트 수정
- WSL 환경에서 이미지 붙여넣기·Windows 11 스크린샷 붙여넣기 수정 및 Windows Explorer 이미지 드래그 지원 추가; 장문 대화 렌더링 성능 개선
