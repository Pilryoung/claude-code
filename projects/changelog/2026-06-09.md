# Claude Code 신규 패치 — 2026-06-09

## v2.1.169 (released 2026-06-09)
- `--safe-mode` 플래그 및 `CLAUDE_CODE_SAFE_MODE` 환경변수 추가 — CLAUDE.md, 플러그인, 훅, MCP 서버 등 모든 커스터마이제이션을 비활성화한 채로 시작해 트러블슈팅에 활용 가능
- `/cd` 명령어 추가 — 프롬프트 캐시를 끊지 않고 세션의 작업 디렉터리를 변경할 수 있음
- `disableBundledSkills` 설정 및 `CLAUDE_CODE_DISABLE_BUNDLED_SKILLS` 환경변수 추가 — 번들 스킬·워크플로우·내장 슬래시 명령어를 모델에서 숨기기 가능
- macOS에서 각 턴 시작 시 발생하던 ~30-50ms UI 지연 수정, 응답 스트리밍·스피너 애니메이션 중 CPU 사용량 감소
- 엔터프라이즈 관리형 MCP 정책(`allowedMcpServers`/`deniedMcpServers`)이 재연결·IDE 설정·첫 세션에서 적용되지 않던 버그 및 원격 설정 없는 조직의 느린 콜드 스타트 수정
- Windows에서 `claude -p` 실행 시 느리거나 멈추는 버그 수정 (v2.1.161 회귀), 백그라운드 에이전트가 프로젝트 레벨 `env` 값을 무시하던 버그 수정
