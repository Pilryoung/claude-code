# Claude Code 신규 패치 — 2026-07-22

## v2.1.217 (released 2026-07-21)
- 프롬프트 입력창에 이모지 단축코드 자동완성 추가 (`:heart:` → ❤️, `:hea` 입력 시 제안 팝업 / `emojiCompletionEnabled` 설정으로 비활성화 가능)
- 트랜스크립트 쓰기 실패(디스크 풀, 세션 저장 비활성화 등) 시 무음 손실 대신 경고 표시
- MCP 도구 출력이 잘릴 때 전체 미절단 결과가 세션 종료까지 메모리에 남던 누수 수정
- Windows 자동 업데이트 실패 시 `claude.exe`가 사라지는 문제 수정 — 실패 시 백업 실행 파일 자동 복원
- 백그라운드 세션이 심볼릭 링크 작업 디렉터리를 정규화하지 않아 워크스페이스 폴더를 벗어날 수 있던 격리 버그 수정
- Bedrock의 Claude Opus 4.8에서 auto-compact가 트리거되지 않고 한도 초과 시 `/compact`가 실패하던 버그 수정
- Claude Desktop 세션에서 mTLS·TLS 검증·OAuth 범위·프록시 설정이 무시되던 버그 수정
- 서브에이전트 동시 실행 수 상한 추가 (기본 20, `CLAUDE_CODE_MAX_CONCURRENT_SUBAGENTS`로 조정) — 메시지 하나가 무한 백그라운드 에이전트 생성하는 상황 방지
- 서브에이전트가 기본적으로 하위 서브에이전트를 생성하지 않도록 변경 (`CLAUDE_CODE_MAX_SUBAGENT_SPAWN_DEPTH`로 허용 깊이 조정 가능)
- `--max-budget-usd` 한도 초과 시 백그라운드 서브에이전트도 중단되도록 수정
