# Claude Code 신규 패치 — 2026-06-18

## v2.1.181 (released 2026-06-18)
- `/config key=value` 구문 추가 — 대화창에서 바로 설정 변경 가능 (예: `/config thinking=false`), `-p` 및 Remote Control에서도 동작
- Write/Edit이 네트워크 드라이브 및 클라우드 동기 폴더에서 0바이트·잘린 파일을 생성하던 버그 수정
- 긴 문단 스트리밍 개선 — 첫 줄바꿈을 기다리지 않고 텍스트를 한 줄씩 실시간 표시
- 시작 속도 개선 — MCP 서버 없는 환경에서 ~120ms 지연 제거, 느린 네트워크에서 최대 15초 블로킹 문제 수정
- 서브에이전트 패널 개선 및 전방향 서브에이전트에도 5단계 깊이 제한 적용, `ANTHROPIC_BASE_URL`·Foundry에서 프롬프트 캐싱 미작동 버그 수정
