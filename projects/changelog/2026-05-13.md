# Claude Code 신규 패치 — 2026-05-13

## v2.1.140 (released 2026-05-12)
- Agent tool `subagent_type` 값의 대소문자·구분자 무관 매칭 지원 (`"Code Reviewer"` → `code-reviewer` 자동 해석)
- `/goal` 명령어가 `disableAllHooks`·`allowManagedHooksOnly` 설정 환경에서 무한 대기하던 버그 수정 → 명확한 오류 메시지 출력
- `claude --bg` 백그라운드 서비스가 유휴 종료 직전에 "connection dropped mid-request" 오류를 반환하던 버그 수정
- 심볼릭 링크된 설정 파일의 hot-reload 시 이벤트 오귀속 및 오작동하는 `ConfigChange` 훅 발생 버그 수정
- Windows에서 `gh` 등 누락 실행파일이 `where.exe`를 매번 동기 호출해 이벤트 루프가 멈추던 문제 수정
